我正在开发一套基于单目深度估计模型Depth Anything 3；特征匹配模型RoMa v2；分割大模型Segment Anything 3的单目结构三维位移测量系统。三个模型的源代码分别放置在了对应的子文件夹中。我目前的测试用视频数据共有1501帧，视频中包含2个作为参考点的的Aruco码，以使relative depth换算为metric depth。
我先为你阐述我目前的系统设计，你先不需要立刻开始为我编码，而是与我讨论整套系统的整体框架与细节，我能考虑到的系统模块包括相机标定与校正（棋盘格）、Aruco码识别与z轴坐标计算、平面内位移计算、平面外位移计算、三个模型调用与结果输出。
在我们确认系统设计已经完备的情况下再开始编码，接下来我为你阐述我规划的系统大体结构与主要功能：
首先会使用Segment Anything 3，通过第一帧及后续校正帧，实现few-shot生成所有图像感兴趣区域的mask，用于后续提示Roma v2和Depth anything 3，使这两个模型只关注mask区域内的要测量的目标物体。我已经编写好了调用的代码并成功输出了预期的结果，也就是文件夹中的sam3_promptmask.py。调用代码如下
python sam3_promptmask.py \
  --input-mode frames \
  --frames-dir /lustre/home/acct-cwj/cwj-user1/sam3-main/assets/videos/3072-3072 \
  --mask-path /lustre/home/acct-cwj/cwj-user1/sam3-main/assets/videos/mask-instance-3072/frame_000501.png \
  --correction-mask-dir /lustre/home/acct-cwj/cwj-user1/sam3-main/assets/videos/mask-instance-3072 \
  --correction-frame-numbers 601,701,801,901,1001,1101,1201,1301,1401,1501,1601,1701,1801,1851,1901,1911,1921,1931,1941,1951,1961,1971,1981,1991 \
  --mask-mode label_id \
  --target-mask-ids 1,2,3 \
  --target-names beanstalk,aruco1,aruco2 \
  --output-dir /lustre/home/acct-cwj/cwj-user1/sam3-main/assets/videos/output

然后特征匹配模型RoMa v2用于测量平面内位移（结合相机内参，内参通过棋盘格标定板标定）。其中调用RoMa v2进行长时间特征点追踪的代码我已经写出并成功输出了预期结果，也就是文件long_time_track.py。调用其运行的指令为  
python test.py \
  --checkpoint_path /lustre/home/acct-cwj/cwj-user1/RoMaV2-main/weights/romav2.0.1.pt \
  --start_frame 501 \
  --end_frame 2001 \
  --num_points 20000 \
  --correction_interval 25 \
  --output_dir output1 \
  --top_video_points 32 \
  --video_fps 20 \
  --video_max_side 1280 \
  --save_npz
其中得到的有效点数（在长时间追踪中输出符合逾期的完整坐标变化曲线的点）为200个，通过特征点的坐标变化结合相机内参可以计算得到平面内（x、y）的特征点位移变化。

随后单目深度估计模型Depth Anything 3通过mask prompt的方式（mask由前述sam 3生成），生成出需要测量的目标物体区域的相对深度。调用Depth Anything 3的代码我已经写出并成功输出了预期结果，也就是infer_depth_sequence_png.py。调用其运行的指令为
python infer_depth_sequence_png.py \
    --input-dir /lustre/home/acct-cwj/cwj-user1/sam3-main/assets/videos/3072-3072 \
    --mask-dir /lustre/home/acct-cwj/cwj-user1/sam3-main/assets/videos/output/masks/combined_instance \
    --output-dir /lustre/home/acct-cwj/cwj-user1/DA3-main/output1 \
    --model-dir depth-anything/DA3-GIANT-1.1 \
    --mode temporal_window \
    --roi-mode fused_masked_affine \
    --fusion-alpha 0.6 \
    --window-size 8 \
    --overlap 4 \
    --process-res 504


综合以上三个模型的运行结果，可以计算得到每个特征点的x、y、z坐标随时间的变化。其中x、y平面内坐标通过相机内参结合特征点追踪计算获得，平面外z坐标通过单目深度估计结合Aruco码计算获得。
